# Mata App - Features Summary

**Mata** is a mobile marketplace application for bartering and swapping items. The concept is "value for value" trading instead of "value for money", promoting sustainable reuse of goods and fostering community connections.

---

## Core Features

### 1. Item Management

- **Add Items**: Create listings with images, name, description, category, condition, swap type, and location
- **Edit Items**: Modify existing listings
- **Delete Items**: Remove items with confirmation for pending deals
- **Draft Mode**: Save items as drafts before publishing
- **Image Gallery**: Multiple images per item with camera/gallery picker
- **Item Conditions**: Define item condition (new, used, with issues, etc.)
- **Archived Items**: View previously swapped/inactive items

### 2. Deals & Trading System

- **Swap Offers**: Send offers to trade items with other users
- **Incoming Deals**: Manage swap requests received from others
- **Outgoing Deals**: Track offers sent to other users
- **Deal Status Tracking**: New, Accepted/Started, Closed, Rejected, Cancelled
- **Deal Confirmation**: Close deals when exchange is complete
- **Archived Deals**: History of completed and past deals
- **Rating System**: Rate users after successful deals

### 3. Discovery & Search

- **Nearby Items**: Location-based item discovery
- **Recommended Items**: Personalized suggestions based on user interests
- **Advanced Search**: Filter by category, country, state, city, swap type, and condition
- **Categories**: Browse items by category hierarchy (multi-level)
- **Free Items**: Special filter for items offered for free

### 4. User Profiles

- **User Registration**: Email-based signup with validation
- **Social Login**: Facebook and Apple authentication
- **Guest Mode**: Browse without registration
- **Profile Management**: Edit name, phone, location, and preferences
- **Change Password**: Secure password updates
- **Profile Privacy**: Control profile visibility to others
- **User Interests**: Set category preferences for recommendations
- **Marketing Preferences**: Opt-in/out for promotional messages

### 5. Wishlist

- **Favorite Items**: Save items to wishlist for later
- **Wishlist Management**: View and manage saved items

### 6. Notifications

- **Push Notifications**: Real-time alerts for deals, messages, etc.
- **Notification Center**: View all notifications in-app

### 7. Chat / Messaging

- **In-App Chat**: Direct messaging with other users
- **Deal-Based Chat**: Communication within deal context

### 8. Location Services

- **Location Detection**: Auto-detect user location
- **Location Selection**: Manual location picker with Google Places
- **Map View**: View items on a map

## Swap Types

The app supports different types of swaps:

- **Barter/Trade**: Exchange item for item
- **Free**: Give away items for free
- **Service Exchange**: Trade services

---

## User Roles

1. **Guest Users**: Browse items, limited functionality
2. **Registered Users**: Full access to all features
3. **Item Owners**: Manage their own listings
4. **Admins** (implied): Block/unblock items functionality
